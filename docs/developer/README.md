## Architecture Overview

```
dc29/
├── protocol.py        Pure-Python protocol constants — no I/O, no dependencies
├── badge.py           BadgeAPI — thread-safe serial interface
├── config.py          TOML config (~/.config/dc29/config.toml)
├── cli.py             Typer CLI (dc29 entry point)
├── bridges/
│   ├── base.py        BaseBridge ABC + BridgePage / PageButton models
│   ├── teams.py       TeamsBridge — Microsoft Teams Local API
│   ├── slack.py       SlackBridge — Slack focus detection + shortcuts
│   ├── outlook.py     OutlookBridge — Outlook focus detection + shortcuts
│   ├── focus.py       FocusBridge — shared focus detection with TTL cache
│   ├── generic.py     GenericFocusBridge — app pages from the registry
│   └── registry.py    ALL_PAGES — registry of all generic app bridge pages
└── tui/
    └── app.py         BadgeTUI — Textual terminal UI
```

The badge communicates over USB CDC serial at 9600 baud using a binary escape-byte protocol. `BadgeAPI` manages the serial port and dispatches events. Bridges subclass `BaseBridge` to connect external services and control badge LEDs.

Full protocol specification: [docs/spine/01-protocol.md](../spine/01-protocol.md)

Full architecture description: [docs/spine/02-architecture.md](../spine/02-architecture.md)

## Thread Model

`BadgeAPI` runs a **daemon reader thread** (blocking `serial.read(1)` loop). The application typically runs an **asyncio event loop** in the main thread. Communication between them uses `loop.call_soon_threadsafe(queue.put_nowait, value)`.

```
Main thread (asyncio)
├── bridge.run() coroutines (Teams, Slack, Outlook, Generic×N)
└── BadgeTUI (optional) — Textual rendering

BadgeAPI reader thread (daemon)
└── _reader_loop() → _parse_rx() → _dispatch_rx()
    └── on_button_press, on_key_reply, on_state_change, ...
        All fired from the reader thread.
```

## Quick Start: Send a Command

```python
from dc29.badge import BadgeAPI
from dc29.protocol import MuteState

badge = BadgeAPI("/dev/tty.usbmodem14201", brightness=0.8)

badge.set_mute_state(MuteState.MUTED)    # LED 4 red
badge.set_mute_state(MuteState.UNMUTED)  # LED 4 green
badge.set_led(2, 0, 100, 255)           # LED 2 blue

badge.close()
```

For the full Teams integration, see [building-bridges.md](building-bridges.md) and `dc29/bridges/teams.py`.
