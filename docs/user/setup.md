## Requirements

- DC29 badge with USB cable
- Python 3.10 or newer
- macOS (primary) or Windows

---

## Step 1: Install

```bash
pip install "dc29-badge[tui,hotkey]"
```

Verify:

```bash
dc29 --help
```

---

## Step 2: Connect the badge

1. Plug the badge into USB
2. Find the serial port:

   **macOS:**
   ```bash
   ls /dev/tty.usbmodem*
   ```

   **Linux:**
   ```bash
   ls /dev/ttyACM*
   ```

   **Windows:** Device Manager → Ports (COM & LPT) — look for USB Serial Device

The port suffix changes between USB ports and reboots on macOS — re-check after replug.

> **Windows only:** If no port appears, install the CDC driver: Device Manager → unknown device → Update Driver → browse to `Firmware/Source/DC29/atmel_devices_cdc.inf`.

---

## Step 3: Enable Teams Local API (one time)

Open Microsoft Teams → **Settings** → **Privacy** → **Third-party app API**. Toggle **Enable third-party API** on. Restart Teams if prompted.

If the toggle is missing or greyed out, your IT admin may have disabled it via tenant policy.

---

## Step 4: Run dc29 flow

```bash
dc29 flow
```

Add `-v` for verbose logging:

```bash
dc29 flow -v
```

This starts all bridges: Teams mute indicator, Slack huddle detection, Outlook email shortcuts, and 15 app shortcut pages.

---

## Step 5: Pair with Teams (first time only)

> **Important:** Pairing only works during an active Teams meeting. The authorization dialog does not appear outside of a call.

1. **Kill Elgato Stream Deck** if you use it — it holds the same Teams API port and blocks dc29:
   ```bash
   killall "Stream Deck"
   ```

2. **Delete any stale token:**
   ```bash
   rm ~/.dc29_teams_token
   ```

3. **Remove the device from Teams API settings** if it appears there:
   - Teams → Settings → Privacy → Third-party app API
   - If **DC29: MuteIndicator** is in the Allowed or Blocked list, block it, then remove it entirely

4. **Join a Teams meeting** (Calendar → Meet now for a quick test call)

5. **Run `dc29 flow -v`** and watch for a **"New connection request"** popup in Teams

6. **Click Allow** — the token saves to `~/.dc29_teams_token`. Future runs connect automatically

## What to expect

| Situation | LED 4 |
|-----------|-------|
| Not in a meeting | Off |
| In a meeting, muted | Red |
| In a meeting, unmuted | Green |

When you switch to Outlook, LEDs 1–4 change to Outlook shortcut colors. Same for Slack huddles and other supported apps.

## macOS: Accessibility permission

The optional hotkey feature uses pynput and requires Accessibility access:

1. **System Settings → Privacy & Security → Accessibility**
2. Enable your terminal app (iTerm2, Terminal.app, etc.)
3. Restart the terminal

Without this, the hotkey is silently disabled but everything else works.

## Autostart

```bash
dc29 autostart install   # install login agent
dc29 autostart status    # check if running
dc29 autostart remove    # uninstall
```

Installs a launchd agent (macOS), systemd service (Linux), or Task Scheduler entry (Windows).
