Connect

Uses the Web Serial API to talk to the badge's CDC port. Chrome / Edge on macOS, Windows, or Linux. When you click Connect, the browser shows a port picker — choose the badge's usbmodem* entry.

Vault (F07)

Two slots, each up to 16 keystrokes (≈ 32 plain ASCII chars). Plaintext EEPROM — never use for real credentials.

Stay Awake (F08)

One pulse keeps macOS awake for ~10 s; autonomous mode pulses every 30 s for the duration you set.

LEDs

Click a swatch to set that LED to a color immediately. RAM-only — power-cycle reverts to the EEPROM defaults.

Buzzer + button feedback

TOTP (F09)

RFC 6238 6-digit TOTP, 30 s window. Provision pushes the base32-decoded raw key to the badge's EEPROM (1 slot). Fire syncs the host clock to the badge then types the current code into the focused window — there's a 3 s countdown so you can switch tabs. Plaintext EEPROM — never use for high-stakes accounts.

Per-button keymap

What each badge button types via HID-Kbd. Each button stores a single (modifier, key) pair in EEPROM. Modifier checkboxes map to the LEFT side of each modifier (Ctrl=0x01, Shift=0x02, Alt=0x04, Cmd=0x08); right-side modifiers are preserved if the badge already had them set. Key field accepts a single char, a special name (Enter / Tab / Esc / Space / Up / F1…F12), a media key (media:mute / volup / voldown / play), or a hex literal (0x04).

WLED knobs

Speed / intensity / palette controls used by the WLED-ported effect modes (firmware effect IDs ≥ 19). Effects in the hand-rolled range (1–18) ignore the palette but still respect speed + intensity.

F01/F02 modifier actions

Assign double-tap / triple-tap / long-press / chord actions per button. Same key syntax as the per-button keymap above. RAM-only on the firmware — actions clear on every power cycle, so re-apply after any unplug.

Chord shortcuts — two buttons pressed within 80 ms.

Macro recorder

Click into the textbox below and type whatever keystroke sequence you want. Modifiers are captured (Ctrl / Shift / Alt / Cmd). Save to a vault slot when done — vault slots hold up to 16 keystrokes. Browser-level shortcuts (Cmd+Q, Cmd+W, etc.) can't be intercepted; if you need those in your macro, use "Type any string" or hand-write the keymap entry.

Share / load config

Encodes current LED swatches + WLED knobs into a URL hash. Send the link to a friend; when they open it (and Connect), the page offers to apply the shared config. Vault contents are never included (those are sensitive and stay on your badge).

Type any string (F06)

Sends arbitrary text to the focused window via the HID burst path. Auto-chunks payloads >256 (mod, key) pairs. 3 s countdown so you can switch tabs before keystrokes land.

Effect mode

Firmware-driven LED effects. Picks one and sends it; the badge keeps it running until you switch. Off hands LEDs back to whatever's painting them (manual swatch / Stay Awake).

Recent activity

Live button-press + modifier events from the badge. Useful while configuring keymaps / F01 modifier-table entries. Most recent at the top; capped at 50 entries.
