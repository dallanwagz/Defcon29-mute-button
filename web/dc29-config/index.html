Connect

Uses the Web Serial API to talk to the badge's CDC port. Chrome / Edge on macOS, Windows, or Linux. When you click Connect, the browser shows a port picker — choose the badge's usbmodem* entry.

Vault (F07)

Two slots, each up to 16 keystrokes (≈ 32 plain ASCII chars). Plaintext EEPROM — never use for real credentials.

Stay Awake (F08)

One pulse keeps macOS awake for ~10 s; autonomous mode pulses every 30 s for the duration you set.

LEDs

Click a swatch to set that LED to a color immediately. RAM-only — power-cycle reverts to the EEPROM defaults.
